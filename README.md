A high-performance, lightweight **Interactive Neural Network Cyber Animation** built entirely in vanilla JavaScript and HTML5 Canvas. Perfect for AI startups, cybersecurity landing pages, international policy dashboards, or any premium technology website looking for an immersive, modern aesthetic without the heavy load of external libraries like Three.js.

## ✨ Features

- **Blazing Fast Performance**: Uses `{ alpha: false }` for bypassing expensive transparency composites. Dynamically caps node density based on the user's viewport resolution to guarantee 60fps across all devices.
- **Interactive Cyber-Nodes**: Nodes gently float and establish glowing connection lines when in proximity to each other.
- **Mouse Repulsion & Attraction**: Move your cursor to physically push nodes away, while simultaneously drawing targeted, cyber-violet data connection lines directly to your mouse pointer.
- **Zero Dependencies**: Pure Vanilla JS, HTML, and CSS. No React, no Three.js, no jQuery. Drop it into any stack instantly.
- **Accessibility Friendly**: Automatically detects `prefers-reduced-motion` and disables the animation if requested by the user's OS settings.

## 🚀 How to Use

Drop the code directly into any webpage. 

1. Create a `<canvas id="cyber-canvas"></canvas>` element on your page.
2. Ensure the canvas is styled to cover the background (e.g., `position: absolute; top: 0; left: 0; width: 100%; height: 100%; z-index: 1;`).
3. Include the JS script found in `index.html`.

You can easily tweak the variables inside the script:
- `D = 150` — Maximum distance for nodes to connect to each other.
- `fillStyle = 'rgba(56,189,248,0.8)'` — Customize the cyan glow color of the nodes.
- `strokeStyle = 'rgba(139,92,246...'` — Customize the violet connection color to the user's mouse.
